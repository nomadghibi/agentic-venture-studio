# Founder Wedge Playbook

## Wedge Positioning
Product brand: **Agentic Venture Studio**  
Commercial wedge: **AI Venture Validation Workspace for Founders**

One-line promise:
Validate startup ideas before you waste weeks building them.

## Target ICP
- Solo founders building SaaS.
- Two to five person founding teams.
- AI-native builders with many ideas and weak prioritization structure.
- Buyers willing to pay for speed, clarity, and risk reduction.

## Core Workflow To Sell
1. Idea intake
2. Market pain validation
3. Feasibility scoring
4. Monetization analysis
5. PRD generation
6. Architecture + MVP roadmap
7. Build / pause / kill recommendation

## Pricing Hypothesis
- Starter: $29/month
- Pro: $79/month
- Studio: $199/month

Fallback if retention is weak:
- Credit model (pay per full validation run)

## Founder Demo Script (8-10 minutes)
1. Show a raw idea submitted in the workspace.
2. Run validation and scoring pass.
3. Review monetization and feasibility outputs.
4. Generate PRD + architecture + MVP plan.
5. Show final recommendation: build, pause, or kill.
6. Show artifact history and why this is better than ad-hoc chat.
7. Close with pricing and ask for a paid pilot.

## Interview Questions (10-15 users)
1. What result would make this worth paying for?
2. Which output matters most: validation, feasibility, monetization, PRD, or roadmap?
3. Would you pay monthly or per report?
4. What makes this more useful than using ChatGPT manually?
5. What trust signals do you need before acting on outputs?
6. Would you use this once or repeatedly each month?
7. What would block adoption?

## 4-Week Execution
- Week 1: tighten messaging and demo.
- Week 2: run founder calls and collect objections.
- Week 3: cut low-value features and improve trust signals.
- Week 4: close first paid users.
