# MVP_ROADMAP.md

## Phase 0: Foundation
### Goal
Stand up the core operating system for opportunity discovery and evaluation.

### Deliverables
- auth and roles
- opportunity entity model
- signal ingestion pipeline
- scoring framework
- agent runtime foundation
- approval workflow
- audit logging
- founder dashboard skeleton

### Exit Criteria
- can ingest signals
- can create opportunity records
- can route through validation stages

## Phase 1: Discovery and Validation MVP
### Goal
Prove the system can find and rank build-worthy opportunities.

### Deliverables
- Opportunity Discovery Agent
- Market Research Agent
- Problem Validation Agent
- Buildability Agent
- Monetization Agent
- opportunity workspace UI
- evidence-linked reports
- gate-based progression
- knowledge wiki generation
- PRD generator

### KPIs
- time from signal to scored opportunity
- reviewer agreement rate
- number of high-quality opportunities per month

## Phase 2: Product Design System
### Goal
Turn approved opportunities into build-ready product artifacts.

### Deliverables
- Product Strategy Agent
- Architecture Agent
- workflows generator
- prompt contracts generator
- roadmap generator
- artifact versioning
- approval UI for build readiness

## Phase 3: Build Assist
### Goal
Support accelerated MVP creation.

### Deliverables
- Builder Agent
- repo scaffold templates
- code generation workflows
- schema generation
- test generation
- CI recommendations

## Phase 4: Launch Assist
### Goal
Support first-market validation.

### Deliverables
- Launch Agent
- positioning assets
- landing page copy
- KPI instrumentation templates
- launch checklists
- experiment tracker

## Phase 5: Portfolio Intelligence
### Goal
Operate multiple ventures with disciplined review.

### Deliverables
- Revenue Intelligence Agent
- Portfolio Manager Agent
- venture dashboard
- cost/revenue comparison
- scale/pause/kill recommendations

## Recommended MVP Cut
Start with:
- Discovery Agent
- Validation Agent
- Buildability Agent
- Monetization Agent
- Product Strategy Agent
- Knowledge Librarian Agent
- dashboard
- approvals
- scoring
- artifact generation

Do not start with:
- full autonomous build/deploy
- autonomous paid ads
- self-modifying prompts
- fully automated capital allocation

## Initial Sector Recommendation
For first traction, focus on one of:
- home services SaaS
- local business automation
- AI front-office products
- SMB workflow software
